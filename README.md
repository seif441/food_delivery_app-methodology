# Food Delivery App

A full-stack food delivery system built using:

- Spring Boot
- SQL Server
- Docker
- HTML/CSS/JavaScript

## Features

- Customer ordering
- Cart system
- Admin dashboard
- Delivery dashboard
- Order tracking

## Authentication Module
Implemented login and registration system using JWT authentication.
